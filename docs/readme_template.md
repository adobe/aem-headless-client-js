# AEM Headless SDK

### Installing

```bash
$ npm install @adobe/aem-headless-sdk
```

### Usage
Import functions from the package
```javascript
const {
  postQuery,
  saveQuery,
  getQuery,
  listQueries
} = require('@adobe/aem-headless-sdk');
```
